API: кредиты вклады


Параметы post запроса для кредита:


  • Адрес: {{MAIN_URL}}api/calc_deposit

  • "value": сумма кредита (int)

  • "percent": процент (float)

  • "loan_time": срок кредита в месяцах (int)

  • "pay_type": тип кредита (аннуитетный/дифференцированный)

  • "loan_date": дата выдачи (год-месяц-день)


Пример запроса:

from requests import post

request = post('{{MAIN_URL}}api/calc_loan', json={
"value": 1000000,
"percent": 5.00,
"loan_time": 36,
"pay_type": "аннуитетный",
"loan_date": "2024-04-21" }).json()
